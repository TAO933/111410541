# 骰子大小遊戲 - 作業說明

## 程式簡介

兩位玩家（Player 1 與 Player 2）各自隨機擲出一顆骰子（1~6 點），系統根據點數判定勝負，並顯示對應的骰子圖片和遊戲結果。

```xaml.cs
 private void BtnRoll_Click(object sender, RoutedEventArgs e)
 {
     // B. 隨機骰子和動態化骰子圖片：根據生成的隨機數，動態載入對應的骰子圖片
     int player1Value = _random.Next(1, 7); // 產生 1~6 的隨機整數
     int player2Value = _random.Next(1, 7); // 產生 1~6 的隨機整數

     // 更新兩位玩家的骰子圖片
     LoadDiceImage(imgPlayer1, player1Value);
     LoadDiceImage(imgPlayer2, player2Value);

     // 更新點數顯示文字
     txtPlayer1Score.Text = $"點數：{player1Value}";
     txtPlayer2Score.Text = $"點數：{player2Value}";

     // C. 勝負判定：比較兩方點數，更新遊戲結果
     DetermineWinner(player1Value, player2Value);
 }
```

**載入骰子圖片到指定的 Image 控制項**
```xaml.cs
 private void LoadDiceImage(System.Windows.Controls.Image imageControl, int diceValue)
{
    try
    {
        // 1. 取得程式執行檔所在的資料夾路徑
        string baseDirectory = AppDomain.CurrentDomain.BaseDirectory;

        // 2. 組合相對路徑：執行目錄下的 image 子資料夾
        // 假設你的圖片放在 bin\Debug\net10.0-windows\image\ 
        string imagePath = System.IO.Path.Combine(baseDirectory, "image", $"{diceValue}.png");

        // 檢查檔案是否存在（增加程式穩定性）
        if (!System.IO.File.Exists(imagePath))
        {
            throw new System.IO.FileNotFoundException($"找不到圖片檔案：{imagePath}");
        }

        // 3. 建立圖片並顯示
        BitmapImage bitmap = new BitmapImage();
        bitmap.BeginInit();
        bitmap.UriSource = new Uri(imagePath, UriKind.Absolute);
        bitmap.CacheOption = BitmapCacheOption.OnLoad;
        bitmap.EndInit();

        imageControl.Source = bitmap;
    }
    catch (Exception ex)
    {
        MessageBox.Show($"圖片載入出錯：{ex.Message}");
    }
}
```

**判斷勝負並更新結果顯示**
```xaml.cs
         private void DetermineWinner(int p1, int p2)
        {
            if (p1 > p2)
            {
                // Player 1 獲勝
                txtResult.Text = "🏆 Player 1 win！";
                txtResult.Foreground = new SolidColorBrush(Color.FromRgb(100, 220, 130)); // 綠色
            }
            else if (p1 < p2)
            {
                // Player 2 獲勝
                txtResult.Text = "🏆 Player 2 win！";
                txtResult.Foreground = new SolidColorBrush(Color.FromRgb(100, 180, 255)); // 藍色
            }
            else
            {
                // 平手
                txtResult.Text = "🤝 Tie！";
                txtResult.Foreground = new SolidColorBrush(Color.FromRgb(255, 214, 165)); // 橘色
            }
        }

```
###  勝負判定

- **作法**：在 `DetermineWinner()` 方法中比較兩方點數，更新 `txtResult` 文字方塊的內容與顏色。
  - 若 Player 1 > Player 2 → 顯示「🏆 Player 1 win！」（綠色）
  - 若 Player 1 < Player 2 → 顯示「🏆 Player 2 win！」（藍色）
  - 若平手 → 顯示「🤝 Tie！」（橘色）
