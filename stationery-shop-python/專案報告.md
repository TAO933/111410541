# 一頁式文具店 (Python 版) - 專案報告

## 1. 專案簡介
本專案為一個「高質感、一頁式」的線上文具店。採用 **Python Flask** 作為後端架構，並搭配輕量級的 **SQLite** 資料庫與純前端技術（HTML/CSS/JS），實現免切換頁面（SPA 模式）的流暢購物、下單及留言回覆體驗。

此版本移除了對 Node.js / npm 的依賴，使得在僅安裝 Python 的本地端電腦上也能輕鬆運行與開發。

---

## 2. 系統架構與技術選型
*   **後端框架：** Python Flask (v3.1.3+)
    *   **選擇原因：** 輕量級、啟動快速，非常適合用來建立 API 端點並託管靜態前端檔案。
*   **資料庫：** SQLite3 (Python 內建模組)
    *   **選擇原因：** 免安裝獨立伺服器，資料儲存於單一檔案 `stationery.db` 中，完美適合作為本地端開發與教學專案。
*   **前端技術：** Vanilla HTML5、CSS3、Vanilla JavaScript (ES6)
    *   **選擇原因：** 無需任何編譯與打包工具（如 Webpack/Vite），瀏覽器可直接解析執行，極大地降低了環境部署的難度。
*   **介面設計：** 採用現代感的 **Glassmorphism (玻璃擬物化)** 風格，搭配 HSL 漸變色調、細邊框與柔和陰影。

---

## 3. 功能模組說明

### (1) 商品展示 (Product Grid)
*   **動態載入**：首頁載入時，透過前端 `fetch('/api/products')` 獲取後端 SQLite 內的 30 項真實文具資料。
*   **分組展示**：產品依據系列被分為三大類：**書寫與修正系列**、**收納與整理系列**、**膠貼與辦公系列**，呈現整齊且精美的排版。
*   **微動畫**：商品圖片在滑鼠懸停時會平滑放大（Scale Up 1.1x），提升互動感。

### (2) 購物車預覽與結帳 (Order Modal)
*   點擊「查看 / 購買」後，會以不換頁方式開啟具有模糊背景（`backdrop-filter: blur(4px)`）的懸浮視窗。
*   使用者可填寫姓名與數量。系統會根據商品單價與數量，在前端實時計算總金額。
*   點選「確認結帳」後，表單將以 POST 方法傳送至 `/api/orders`，後端將訂單寫入 `orders` 資料表中。

### (3) 評論與回覆留言板 (Comment Section)
*   **讀取與寫入**：在商品詳情彈窗底部，展示針對該商品的顧客評論。
*   **巢狀回覆結構**：支援回覆功能，點擊「回覆」後，可針對特定留言進行回覆。留言在資料庫中透過 `parent_id` 欄位進行關聯，前端會以縮排和不同的主題色（主色 vs 輔色）呈現層級關係。

---

## 4. 資料庫綱要 (Schema)
系統包含三個核心資料表：
1.  **products**: 記錄商品 ID、名稱 (`name`)、價格 (`price`)、敘述 (`description`)、展示圖片 (`image`) 與系列分類 (`series`)。
2.  **orders**: 記錄訂單 ID、購買的商品名稱 (`product_name`)、數量 (`quantity`)、總價 (`total_price`)、買家名稱 (`customer_name`) 與訂單時間。
3.  **comments**: 記錄留言 ID、商品名稱 (`product_name`)、留言者 (`author_name`)、內容 (`content`)、回覆關聯 ID (`parent_id`) 與留言時間。

## 5. 如何在本地端啟動專案
因為本專案使用的是 Python 伺服器，您可以直接在終端機 (Terminal) 中執行下列指令來啟動：

### 步驟 1：進入專案資料夾
```bash
cd "c:\Users\user\Desktop\網頁設計\111410541_project\stationery-shop-python"
```

### 步驟 2：啟動 Python 伺服器
如果您使用的是 Thonny 內建的 Python，請執行：
```bash
C:\Users\user\AppData\Local\Programs\Thonny\python.exe server.py
```
*(如果是已加入 PATH 的系統 Python，可直接輸入 `python server.py`)*

### 步驟 3：訪問網頁
伺服器啟動後，打開瀏覽器並訪問：
```
http://127.0.0.1:5000
```
即可檢視並操作您的線上文具店！

