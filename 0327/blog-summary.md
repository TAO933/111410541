# Blog 系統開發摘要

## 專案資訊

| 項目 | 內容 |
|------|------|
| 專案名稱 | Blog 系統 |
| 開發時間 | 2026年3月27日 |
| 技術棧 | Node.js + Express + SQLite (sql.js) |
| 版本數量 | 2 個（Threads 風格、Facebook 風格） |

---

## 系統架構

```
┌─────────────────────────────────────────┐
│              前端 (HTML/CSS/JS)          │
├─────────────────────────────────────────┤
│         後端 API (Express.js)            │
├─────────────────────────────────────────┤
│           資料庫 (SQLite)                │
└─────────────────────────────────────────┘
```

## 兩個版本比較

### Threads 風格 (blog/)

| 項目 | 說明 |
|------|------|
| 埠口 | 3000 |
| 主題 | 深色模式 |
| 佈局 | 三欄式（左導覽 + 中間內容 + 右側推薦） |
| 特色 | 底部固定導航列 |

### Facebook 風格 (blog 1/)

| 項目 | 說明 |
|------|------|
| 埠口 | 3001 |
| 主題 | 淺色模式（藍白配色） |
| 佈局 | 三欄式（左功能表 + 中間動態 + 右側工具） |
| 特色 | 限時動態、聯絡人列表 |

## 功能總覽

### 核心功能

| 功能 | 說明 |
|------|------|
| 發布文章 | 輸入文字內容，按 Enter 發布 |
| 個人/公開切換 | 選擇文章可見範圍 |
| 刪除文章 | 個人文章可刪除 |
| 按讚 | 前端互動顯示愛心 |

### 介面功能

| 功能 | 說明 |
|------|------|
| 響應式設計 | 支援手機、平板、桌面 |
| 搜尋框 | 介面元素（尚無實際功能） |
| 登入/註冊 | 介面按鈕（尚無實際功能） |

---

## 資料庫結構

### posts 資料表

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵，自動遞增 |
| title | TEXT | 標題（目前未使用） |
| content | TEXT | 文章內容 |
| author | TEXT | 作者名稱 |
| is_public | INTEGER | 0=個人，1=公開 |
| likes | INTEGER | 按讚數 |
| created_at | DATETIME | 建立時間 |

### profile 資料表

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵 |
| username | TEXT | 使用者名稱 |
| bio | TEXT | 個人簡介 |
| avatar_color | TEXT | 頭像顏色 |

---

## API 端點

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | / | 取得首頁 HTML |
| POST | /api/posts | 發布新文章 |
| DELETE | /api/posts/:id | 刪除指定文章 |

## 啟動方式

```bash
# Threads 版本
cd blog
npm install
npm start
# 瀏覽器開啟 http://localhost:3000

# Facebook 版本
cd "blog 1"
npm install
npm start
# 瀏覽器開啟 http://localhost:3001
```

## 技術選擇理由

| 選擇 | 理由 |
|------|------|
| sql.js | 避免 native module 編譯問題，純 JS 實現 |
| Express | 輕量、簡單、符合需求 |
| 內嵌 HTML | 減少檔案數量，開發快速 |
